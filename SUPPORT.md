# Support

## Where to get help

- **Bug reports / feature requests**: GitHub Issues
- **Questions / usage help**: GitHub Discussions (if enabled). If Discussions are not enabled, open an Issue with the label `question`.

## What not to do

- Please do not send direct messages (DMs) to maintainers for support.
- Please do not post secrets, tokens, or private data in public issues.
